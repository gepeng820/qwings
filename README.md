## git操作

// 先用dev开发  阶段测试完后合并到master

```
1. git clone xxx--项目目录

2. 切换dev分支
   git checkout dev

3. 更新
   git pull

4. git add .

5. git commit -m 'xxx'

6. git push origin dev

```

## 开发流程

```
npm install

npm run dev-node

npm run dev
```

## 文件存放约定

- 公共组件--->components
- 主要内容--->container
- html模版--->page
- ...

## 代码规范约定

- 主页面路由放在`app/index.js`内
- 代码只增不减
- **个人代码须标注个人姓名缩写**
- 引入新模块及时通知组内成员
